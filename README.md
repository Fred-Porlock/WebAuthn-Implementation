## 前端 | Frontend
安装依赖 | install modules
```powershell
npm install
```

运行前端 | run frontend
```powershell
npm run dev
```

## 后端 | Backend
创建并激活虚拟环境 | create and activate virtual environment
```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

安装webauthn和flask库 | install webauthn and flask
```powershell
pip install flask webauthn
```

运行后端app.py | run app.py
```powershell
flask run
```
